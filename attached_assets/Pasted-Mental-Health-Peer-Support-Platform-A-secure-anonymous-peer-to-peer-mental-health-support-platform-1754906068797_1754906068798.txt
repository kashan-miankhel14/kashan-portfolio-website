Mental Health Peer Support Platform
A secure, anonymous peer-to-peer mental health support platform that connects individuals facing similar challenges. Built with PHP and MySQL, featuring encrypted messaging, mood tracking, and professional crisis resources.

Core Functionality
Anonymous User Profiles - Create profiles without revealing personal identity
Smart Peer Matching - Algorithm-based matching with users facing similar challenges
Secure Messaging - End-to-end encrypted chat system between matched peers
Group Support - Join or create support groups for specific mental health topics
Crisis Resources - Immediate access to professional emergency mental health services
Personal Wellness Tools
Mood Tracking - Daily mood logging with triggers and coping strategies
Private Journal - Encrypted personal journal entries
Goal Setting - Set and track mental health goals with progress monitoring
Progress Analytics - Visual representation of mood patterns and goal achievements
Safety & Moderation
AI Chatbot Support - 24/7 automated support and crisis detection
Admin Dashboard - Comprehensive moderation tools and user management
Crisis Alert System - Automatic detection and escalation of emergency situations
Peer Rating System - Community-driven trust and safety mechanisms
ðŸš€ Quick Start
Prerequisites
PHP 7.4 or higher
MySQL 5.7 or higher
Web server (Apache/Nginx)
Modern web browser
Installation
Clone the repository

git clone https://github.com/yourusername/mental-health-peer-support.git
cd mental-health-peer-support
Database Setup

# Create database
mysql -u root -p -e "CREATE DATABASE peer_support_platform;"

# Import schema
mysql -u root -p peer_support_platform < sql/db_schema.sql

# Add admin user (optional)
mysql -u root -p peer_support_platform < sql/add_admin.sql
Configure Database Connection

// Edit src/config.php
define('DB_HOST', 'localhost');
define('DB_NAME', 'peer_support_platform');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
Set Permissions

chmod 755 -R .
chmod 644 src/config.php
Launch Application

Start your web server
Navigate to http://localhost/your-project-folder
Register a new account or use admin login
